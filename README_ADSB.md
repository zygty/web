# 航空轨迹监控系统 - ADSB数据集成（时间轴版本）

## 项目概述

本系统已成功集成了三组ADSB数据文件，支持**时间轴播放**功能。地图上会显示某个时间点正在飞行的航班，你可以通过时间滑块或播放按钮来查看不同时间点的航班分布。

## 核心功能：时间轴控制

- ⏱️ **时间滑块**: 拖动滑块查看任意时间点的航班位置
- ▶️ **自动播放**: 按时间顺序自动播放航班运动
- 🔄 **循环播放**: 到达结束时间后自动从头开始
- ⚡ **速度调节**: 支持0.1x到10x的播放速度
- 📊 **实时统计**: 显示当前时间活跃航班数量

### 3. 使用功能

#### 时间轴控制
- **拖动时间滑块**: 快速跳转到任意时间点
- **查看当前时间**: 顶部时间显示栏显示当前时间位置
- **活跃航班统计**: 实时显示当前时间有多少航班在飞行

#### 播放控制
- **播放/暂停**: 点击播放按钮开始时间轴自动播放
- **调整速度**: 使用速度滑块调整播放速度（0.1x - 10x）
  - 1x速度: 约等于实时100倍速（1小时数据在36秒播放完）
  - 10x速度: 快速浏览整个时间段
  - 0.1x速度: 精细观察某个时间段
- **重置**: 回到起始时间点
- **清除**: 清除所有航班显示

#### 地图交互
- **点击飞机图标**: 查看航班详细信息（机型、高度、速度、航向等）
- **缩放地图**: 滚轮缩放查看不同区域
- **拖动地图**: 拖动查看不同地理区域

#### 航班信息弹窗
点击任意飞机图标会显示：
- 航班号（Callsign）
- 机型
- 注册地（国籍）
- 当前高度（英尺）
- 当前速度（公里/小时）
- 航向角度
- 当前时间
- 经纬度坐标

## 技术实现细节

### 时间轴算法
1. **全局时间范围计算**: 扫描所有航班轨迹点，找出最早和最晚的时间戳
2. **时间插值**: 对于任意给定时间，找到每个航班最接近的轨迹点
3. **动态显示**: 只显示在当前时间前后5分钟内有数据点的航班
4. **平滑动画**: 使用requestAnimationFrame实现流畅的播放效果

### 数据优化
- **简化版数据**: 前100个航班，每个航班最多100个轨迹点
- **时间范围**: 约1.29小时的数据
- **性能优化**: 使用Set数据结构快速判断活跃航班

### 前端技术
- **JavaScript ES6+**: 箭头函数、async/await、模块化
- **Leaflet地图库**: 轻量级开源地图解决方案
- **深色地图主题**: CARTO Dark Matter底图
- **响应式设计**: 适配不同屏幕尺寸
