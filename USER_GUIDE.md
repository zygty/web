# 航空轨迹监控系统 - 快速使用指南

## 一分钟上手

1. **启动服务器**
   ```bash
   cd /Users/liziqi/Desktop/web
   ./start_server.sh
   ```

2. **打开浏览器**
   访问 http://localhost:8000

3. **开始使用**
   - 等待数据加载（约2-3秒）
   - 地图上会显示起始时间的航班
   - 点击"播放"按钮观看时间轴动画

## 主要功能说明

### 🎯 时间轴控制面板

```
┌─────────────────────────────┐
│  时间轴控制                  │
│  ┌─────────────────────┐    │
│  │ 01/01 00:00:00      │    │  ← 当前时间
│  │ 01/01 00:00 - 01:29 │    │  ← 时间范围
│  └─────────────────────┘    │
│  ───────────●─────────       │  ← 时间滑块
│  活跃航班: 45  总航班: 100   │  ← 统计信息
└─────────────────────────────┘
```

**操作方法:**
- 拖动滑块快速跳转到任意时间
- 点击滑块空白处大幅度跳转
- 观察活跃航班数量变化

### 🎮 播放控制面板

```
┌─────────────────────────────┐
│  播放控制                    │
│  [▶ 播放] [⟲ 重置] [✕ 清除]  │  ← 控制按钮
│  播放速度: [1.0]x           │  ← 速度控制
│  ─────────────●───────       │  ← 速度滑块
└─────────────────────────────┘
```

**按钮说明:**
- **▶ 播放** / **⏸ 暂停**: 开始/暂停时间轴自动播放
- **⟲ 重置**: 回到起始时间点
- **✕ 清除**: 清除地图上所有航班

**速度说明:**
- **0.1x**: 慢速播放，适合精细观察
- **1.0x**: 标准速度（约36秒播放1小时数据）
- **10.0x**: 快速浏览，适合查看整体趋势

- 🟢 **绿色渐变圆点内的飞机**: 当前时间正在飞行的航班

## 使用场景

### 场景1: 观察特定时间段的航班分布

1. 拖动时间滑块到目标时间
2. 观察地图上的航班分布
3. 点击感兴趣的飞机查看详情

### 场景2: 追踪单个航班的完整轨迹

1. 找到目标航班并点击
2. 慢速播放（0.5x）观察航班移动
3. 地图会自动跟随选中航班

### 场景3: 分析航班高峰时段

1. 设置慢速播放（0.5x）
2. 观察"活跃航班"数量变化
3. 记录航班数量最多的时间段

### 场景4: 快速浏览整体数据

1. 设置快速播放（5x - 10x）
2. 点击播放按钮
3. 观察航班整体的时空分布模式

## 常见问题

**Q: 为什么有些飞机图标会消失？**
A: 这是正常的。系统只显示当前时间前后5分钟内有数据的航班。时间推移后，一些航班可能已经降落或还未起飞。

**Q: 可以一次显示更多航班吗？**
A: 可以。修改JavaScript中的`maxFlights`变量即可。但要注意性能问题。

**Q: 时间范围是固定的吗？**
A: 不是。系统会自动计算数据中最早和最晚的时间戳，动态设置时间范围。

**Q: 能否导出某个时间点的航班截图？**
A: 可以使用浏览器的截图功能，或者使用系统截图工具。

**Q: 播放速度能否自定义？**
A: 可以。速度滑块范围是0.1x到10x，你可以在JavaScript中调整这个范围。

## 技巧提示

1. **键盘快捷键**: 当前版本主要使用鼠标操作
2. **地图缩放**: 使用鼠标滚轮可以缩放地图查看细节
3. **地图拖动**: 按住鼠标左键拖动地图查看不同区域
4. **双击飞机**: 自动聚焦并缩放到该航班
5. **全屏模式**: 按F11进入全屏获得更好的观看体验
